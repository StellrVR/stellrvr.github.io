F.A.Q.

What is VR Ring?

VR Ring is a webring focused on anything and everything VR! Whether it be PCVR, Quest, Pico, VR supported games, Flat2VR mods. You name it!


What if I don't own a VR headset?

That's okay! As long as you're at least interested in VR, you can join the webring :)


The widget isn't loading properly

Make sure that you copied the code exactly! If you're still having trouble, send me an >EMAIL<.


"This site isn't part of the VRRing webring yet. ..." What is this?

It looks like you're trying to add the webring to your site, but haven't sent an application yet! I have to manually add each website to the directory, so make sure to email me the format listed on the home page.

If you're still getting this after being accepted, then wait a little while for it to update or use Ctrl+R / Cmd+R to refresh your page


Can I still use your VRRing button even though I'm not in the ring?

Absolutely! More promotion is better than none :)
